# Claude Code Easy Suite 使用说明书

欢迎使用 **Claude Code Easy Suite**！本工具专为简化 Anthropic `claude-code` 命令行工具的配置、多项目管理与启动流程而设计。

以下是详细的操作指南：

## 1. 启动与环境检测
首次运行程序时，工具会自动检查您的系统环境：
*   **Node.js**：如果未安装，程序会尝试自动通过 Winget 安装。
*   **Claude Code**：程序会自动安装或更新 `@anthropic-ai/claude-code` 到最新版本。
*   **注意**：如果进行了自动安装，程序可能会自动重启以应用更改。

## 2. 模型配置 (Model Settings)
在使用之前，您需要配置至少一个模型的 API Key。

1.  在主界面 **“模型选择”** 标题的右侧，点击 **“⚙️ 模型设置”** 按钮。
2.  在弹出的窗口中选择您想使用的模型：**GLM**, **Kimi**, **Doubao**, **MiniMax** 或 **Custom**。
3.  **API Key**：在输入框中粘贴您的 API 密钥。如果按钮下方显示**蓝色横线**，表示该模型已成功配置密钥。
4.  **保存**：点击窗口底部的 **“保存并关闭”** 按钮生效。

> **提示**：首次运行且未配置任何 Key 时，系统会自动弹出此窗口。未配置 Key 的模型将无法被选中激活。

## 3. 多项目管理 (Multi-Project)
您可以同时管理多个不同的代码项目，并为每个项目设置独立的路径和模式。

### 3.1 切换项目
*   在 **“Vibe Coding”** 区域可以看到项目选项卡（Tab）。
*   点击对应的项目名称即可快速切换。
*   当项目超过 5 个时，点击 Tab 栏左右两侧的 **“◀”** 或 **“▶”** 箭头进行翻页。

### 3.2 项目管理弹窗
点击 **“Vibe Coding”** 标题右侧的 **“📂 项目管理”** 按钮：
*   **添加项目**：点击“+ 添加新项目”，系统会自动生成一个唯一名称的新项目，默认路径为用户主目录。
*   **重命名**：直接在输入框中修改项目名称。系统会实时检测是否重名，确保每个项目名称唯一。
*   **删除项目**：点击项目右侧的“删除”按钮。
*   **保存**：点击 **“保存并关闭”** 提交所有更改。

## 4. 设置项目参数
在主界面选中某个项目 Tab 后，可以对其进行具体设置：
1.  **项目目录**：显示当前项目的文件夹路径。点击 **“更改”** 重新选择。
2.  **Yolo 模式**：
    *   勾选 **"Yolo Mode"**。
    *   此模式会跳过所有权限询问。**请在信任模型输出的情况下谨慎使用。**

## 5. 启动 Claude Code
1.  确保已选中有效的**模型**（有蓝线标识）和**项目**（有有效路径）。
2.  点击 **“启动 Claude Code”** 按钮。
3.  如果未设置项目目录，底部状态栏将显示红色警告并拦截启动。
4.  成功启动后，将弹出命令行窗口进入交互界面。

## 6. 其他功能
*   **状态显示**：界面最底部会实时显示操作结果（如“保存成功”、“正在同步”等），错误信息将以红色高亮。
*   **多语言切换**：标题栏可切换中（简/繁）、英、日、韩、德、法文。
*   **系统托盘**：右键点击图标可快速切换模型、启动程序或隐藏窗口。
